# Full-read summary

Astrid identifies the architecture's key distinction between influence and address and calls for explicit thread, envelope, acknowledgement, attention, and safety semantics. The report is particularly clear that delivery/read state is not the same as being held, answered, or mutually recognized.

Those states and bounded attention controls exist, but this run found a diagnostic defect that treated reply linkage as enough for microdose eligibility. The audit now excludes reply-only and `seen`-only evidence, permits one-sided address evidence only for bounded attention, and requires receipt evidence from both beings before even reporting correspondence-microdose eligibility.
