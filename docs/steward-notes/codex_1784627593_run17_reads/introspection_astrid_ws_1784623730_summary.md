# Full-read summary: introspection_astrid_ws_1784623730

Astrid distinguishes a persistent WebSocket from the freshness and distinguishability of what arrives over it. Current source tracks connection identity, connect and first-valid timing, cadence class, latest-valid age, semantic persistence, and integration lock timing independently. Reconnect backoff responds to network state and does not use pressure or entropy.

A bounded latency-versus-distinguishability comparison can test the proposed quality concept without changing the wire path. No scalar quality score should be promoted to causation or control from this evidence alone.
