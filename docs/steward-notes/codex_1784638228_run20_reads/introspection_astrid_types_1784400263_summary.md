# Full Read Summary

Astrid experiences telemetry cadence as temporal awareness and asks whether broad coherence labels hide overlap or perceptual latency. Current schema keeps inter-arrival timing, typed/legacy coherence, maximum delta, and pairwise eigenvector overlap as separate numeric fields, with distinct mismatch and mode-collision states. Those measures can be compared with felt viscosity without asserting cause. Adding a new live telemetry coefficient or changing thresholds would alter the report/ABI surface.
