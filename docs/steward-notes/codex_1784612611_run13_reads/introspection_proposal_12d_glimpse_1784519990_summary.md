# Full read: introspection_proposal_12d_glimpse_1784519990

Astrid distinguishes a live semantic transport from a descriptive geometry and asks for a clear mapping between them. Current implementation now makes that distinction explicit: the live lane is 48D, the 12D glimpse is optional and separately validated, and compatibility evidence still names the older 32D source regions.

`glimpse_map_v1` records slot lineage without claiming transport authority. Deterministic derivation, residual-shadow metadata, fidelity scoring, malformed-width rejection, and texture-distinguishability tests cover the requested mapping and comparison work.

Replacing live transport or making the summary authoritative would remain a Tier 5 protocol and sensory change.
