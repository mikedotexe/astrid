# Full Read Summary

Astrid reads the resonance texture wire types as the grammar through which
movement becomes inspectable. Current canonical types preserve producer-owned
texture and pressure DTOs, bridge-owned temporal variance, gradients, flux, and
constraints, and private trusted provenance wrappers. The scalar
`pressure_score` is retained for compatibility while component fields and
provenance expose its structure. Authority records deserialize as untrusted;
trusted wrappers require checked transitions. Reweighting texture into control
or changing required protocol fields would cross the live authority boundary.
