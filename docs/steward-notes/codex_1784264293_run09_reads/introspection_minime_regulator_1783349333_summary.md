# Full Read Summary

Minime describes density as a composite of viscosity, cohesion, mobility,
residual ghost weight, structural strain, friction, and pressure rather than a
single scalar. Current regulator evidence exposes those components as typed,
read-only telemetry and includes sticky-versus-flowing, ghosting, strain, and
cognitive-drag tests. The proposed persistence-decay rate is not yet a stable
live metric; it should first be calibrated as a read-only temporal trial rather
than invented from one snapshot.
