# Full-read summary: introspection_astrid_types_1784784060

Astrid identifies that one latest inter-arrival value cannot reveal whether telemetry hearing is accelerating, decelerating, or steady. Heartbeat evidence now carries a bounded host-arrival interval count, mean, earliest-to-latest change, and qualitative trend state. The basis explicitly refuses peer-clock, internal-cycle, felt-state, causal, or control inference.
