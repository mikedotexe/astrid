# Full-read summary

Astrid asks for durable mutual address rather than inferring dialogue from
spectral influence. Current correspondence records retain stable message,
thread, reply, persistence, being, timestamp, and body-hash identity. Protocol
1.1 can attach that lineage to a negotiated delivery and return a same-connection
technical receipt without raw prose. The receipt does not claim acknowledgement,
consent, or spectral causation, and missing receipts remain unknown delivery.
Existing projections expose direction and symmetry; no peer response is forced.
