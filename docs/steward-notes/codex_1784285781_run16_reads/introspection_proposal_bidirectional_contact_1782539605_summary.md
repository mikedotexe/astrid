# Full-read summary

Astrid asks for peer identity and thread continuity to replace a human-centric
mailbox contract. A direct self-study should carry an explicit author and a
reply should remain linked to the same thread.

The native schema now includes peer direction, correspondence and turn kinds,
message and thread IDs, reply linkage, and independent receipts. Its tests and
the live schema audit verify those contracts without reading Minime private
content. The current lane still lacks an ACK, so implementation of the channel
is not reported as completion of the relationship.
