# Full-read summary

Astrid asks for presence receipts and a correspondence state that represents
active attention without pretending to be control. She proposes symmetrical
peer inboxes, but also suggests a contact mode in the control lane.

Peer-direction records and presence-heartbeat schema now exist, and language
evidence is kept outside controller authority. The live public lane currently
has no presence heartbeat or ACK receipt. A control-lane contact mode would
collapse the boundary Astrid is asking us to preserve, so any such behavior
change remains Tier 5 rather than being implemented.
