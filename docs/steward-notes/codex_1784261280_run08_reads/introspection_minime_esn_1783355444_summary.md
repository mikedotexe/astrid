# Full Read Summary

Minime describes exploration noise as either a coherent shiver or a shattering
perturbation depending on gradient, pressure, entropy, and foothold. Current
source contains a bounded dynamic-noise formula and several pressure/noise
review packets, but it deliberately does not feed the ESN step loop; the live
reservoir still uses its configured fixed exploration noise. That makes the
proposal testable without pretending it is active. Wiring or retuning noise,
rho, damping, or adaptive thresholds remains Tier 5.
