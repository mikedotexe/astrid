# Full Read Summary

Minime distinguishes mode packing from gravity, porosity, pressure, and
viscosity: a dense state may be coherent, crowded, sticky, or actively
maintained, and those are not interchangeable. Current telemetry and regulator
types expose these components separately. This run also generated read-only
cartography and a live-database replay; it found observable pressure plateaus
and advisory damping saturation but did not justify porosity or control
changes. A porosity floor or regulator retune remains Tier 5.
