# Full Read Summary

Astrid distinguishes a working transport from evidence of relationship:
messages can move in both directions while continuity, acknowledgement, and
mutual recognition remain absent. Current correspondence records preserve
thread IDs, native versus legacy provenance, acknowledgement and reply state,
attention lifetime, asymmetry, and explicit
`claimed_pending_native_evidence` status. Recognition remains descriptive
evidence and does not grant runtime, peer-mutation, or control authority.
