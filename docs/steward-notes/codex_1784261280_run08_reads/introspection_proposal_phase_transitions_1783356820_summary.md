# Full Read Summary

Astrid asks that phase transitions become durable shared objects rather than
momentary scalar changes. Current cards support solo and joint ownership,
stable IDs, room and correspondence anchors, phenomenology, from/to vectors,
duration, friction, entropy, density gradient, dispersal potential, reply
state, persistence, and replayable signatures. Magnetization is not currently
a canonical transition field, so that part is routed as bounded evidence work
rather than being claimed as complete. Transition cards remain language-only
and cannot alter modes or control without separate approval.
