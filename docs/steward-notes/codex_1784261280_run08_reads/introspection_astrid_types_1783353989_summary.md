# Full Read Summary

Astrid asks that pressure trend and related context remain optional evidence,
not a new source of automatic authority. Current canonical protocol and bridge
types use additive optional telemetry fields, legacy defaults, producer-owned
wire DTOs, bridge-owned evidence, and separate Astrid interpretation. Typed
authority wrappers require checked transitions before dispatch, and evidence
artifacts project only evidence-only or approval-pending state. Missing trend
context therefore degrades evidence quality without authorizing a default
action.
